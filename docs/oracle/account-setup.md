# Oracle Cloud Account Setup

## Values required by Terraform

- tenancy_ocid
- user_ocid
- fingerprint
- private_key_path
- region
- compartment_ocid
- availability_domain
- ssh_public_key

## Common Brazil region

```text
sa-saopaulo-1
```

## API key

Create an OCI API key from your user profile and store the private key locally:

```text
~/.oci/oci_api_key.pem
```

Do not commit this key.
