# Boom Dev Environment - Oracle Cloud

This environment provisions the first Boom development infrastructure on Oracle Cloud.

## Design

The environment uses reusable modules:

```text
terraform/modules/oracle/network
terraform/modules/oracle/compute
```

Environment-specific configuration is kept in:

```text
terraform/environments/dev/terraform.tfvars
```

This file is generated by:

```bash
make oracle-bootstrap
```

## Commands

From repository root:

```bash
make oracle-bootstrap
make oracle-doctor
make oracle-init
make oracle-plan
make oracle-apply
```

## Current resources

- VCN
- Public subnet
- Internet Gateway
- Route Table
- Security List
- Compute instance
- Cloud-init bootstrap for Docker and Nginx

## Notes

This environment is intentionally simple for the MVP. It is designed to evolve toward:

- remote Terraform state
- private subnet
- object storage backups
- monitoring
- DNS and HTTPS
